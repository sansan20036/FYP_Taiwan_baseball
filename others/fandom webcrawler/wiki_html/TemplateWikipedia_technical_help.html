Usage

This template's initial visibility currently defaults to autocollapse, meaning that if there is another collapsible item on the page (a navbox, sidebar, or table with the collapsible attribute), it is hidden apart from its title bar; if not, it is fully visible.

To change this template's initial visibility, the |state= parameter may be used:

  • {{Wikipedia technical help|state=collapsed}} will show the template collapsed, i.e. hidden apart from its title bar.
  • {{Wikipedia technical help|state=expanded}} will show the template expanded, i.e. fully visible.

Purpose

This is a navbox of the main technical help pages on English Wikipedia.

It doesn't list them all as there are too many, but these should be the most helpful ones of a technical nature.

Additional help

Some of the links are also on the {{Wikipedia help pages}} navbox. Other helpful navboxes for technical matters include:

  • {{Wikipedia templates}} for an index of templates
  • {{Wikipedia referencing}} for referencing templates
  • {{Botnav}} for bot page navigation
  • Wikipedia:WikiProject Accessibility/Navigation menu for accessibility
  • Wikipedia:User scripts/Navbox for user scripts

Also, look at template documentation (such as Template:Navbox/doc), the text of which should also be found in a green section on the template page itself (which for the previous example would be Template:Navbox).

Wikipedia:Categories, lists, and navigation templates is a good comparative overview, and helpful categories themselves include:

  • Category:Wikipedia help and its subcategories such as Category:Wikipedia how-to, Category:Wikipedia image help, Category:Wikipedia interface help, and Category:Wikipedia link help
  • Category:Wikipedia information pages
