Template documentation[view] [edit] [history] [purge]
This template can only be edited by administrators because it is transcluded onto one or more cascade-protected pages.
If you plan to make breaking changes to this template, move it, change scope, or nominate it for deletion or deprecation, please notify the Article Alerts project at Wikipedia talk:Article alerts as a courtesy, as this page is used by the AAlertBot bot to detect and report pages. Thank you!

{{TFAFULL|Bird}} will generate: (Full article...), which is the current preferred linking method to the featured article in TFA blurbs. If it is decided to change this, after appropriate discussion, then a change in the template here will suffice. The template is fully protected from editing because it is continuously used on the Main Page.

As of October 2019, as per discussion at Wikipedia talk:Today's featured article#Featured topics, this template can now take an optional second parameter. For instance, {{TFAFULL|Guy Fawkes|Gunpowder Plot}} now produces: (This article is part of a featured topic: Gunpowder Plot.)

The above documentation is transcluded from Template:TFAFULL/doc. (edit | history)
Editors can experiment in this template's sandbox (edit | diff) and testcases (create) pages.
