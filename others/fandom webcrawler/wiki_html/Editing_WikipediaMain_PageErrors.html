Group notice   Page notice
Please post error reports regarding only what is currently or imminently on the Main Page here, under the corresponding section.
For general discussion about the Main Page, please use Talk:Main Page.

When reporting an error, please be as specific as possible. It can be fixed more quickly when a correction is offered. References are also helpful, especially when reporting obscure factual or grammatical errors; a suggested rewording is helpful.

Please remember that content on the Main Page usually defers to its supporting pages for accuracy or in case of disagreement, so it is normally better to achieve consensus and make any necessary changes there first before reporting it here if necessary.

Once an error has been fixed, rotated off the Main Page or acknowledged not to be an error, the report will be removed from this page; please check the page's revision history for any discussion or action taken. Lengthy discussions should be moved to a suitable location elsewhere.

Please note that placing {{edit fully-protected}} here will not provide a faster response.

You are not logged in. Your IP address will be publicly visible if you make any edits. If you log in or create an account, your edits will be attributed to a username, among other benefits.
Content that violates any copyrights will be deleted. Encyclopedic content must be verifiable through citations to reliable sources.
