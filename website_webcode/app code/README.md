# ProBall Analytics - 專業棒球賽事平台

## 📋 專案概述

ProBall Analytics 是一個現代化的棒球賽事數據分析平台，提供即時賽事資訊、數據分析、AI 預測等功能。

## 🏗️ 專案結構

```
app code/
├── app.html          # 首頁（主頁面）
├── app.css           # 首頁樣式
├── app.js            # 首頁功能
├── matches.html      # 賽事頁面
├── matches.css       # 賽事頁面樣式
├── matches.js        # 賽事頁面功能
└── README.md         # 專案說明
```

## 🎯 主要功能

### 1. 首頁 (app.html)
- **今日焦點賽事**：大圖輪播顯示重點比賽
- **先發投手對決**：投手數據比較與雷達圖
- **關鍵數據比較**：球隊數據對比長條圖
- **AI 模型預測**：勝率預測與詳細報告
- **歷史對戰回顧**：時間軸式對戰記錄
- **主題切換**：日間/夜間模式切換
- **響應式設計**：支援各種螢幕尺寸

### 2. 賽事頁面 (matches.html)
- **Hero 輪播區**：今日重點比賽大圖輪播
- **日期分區導航**：可左右滑動的日期選擇器
- **賽事列表**：詳細的比賽卡片列表
- **即時比分**：進行中比賽的即時比分與進度條
- **功能圖示**：直播、購票、統計等功能按鈕
- **搜尋功能**：球隊/球員/賽事搜尋
- **用戶選單**：個人化設定與收藏

## 📋 使用說明

### 1. 開啟首頁
```bash
# 在瀏覽器中開啟
app code/app.html
```

### 2. 切換到賽事頁面
- 點擊導航列中的「賽事」連結
- 或直接開啟 `matches.html`

### 3. 使用輪播功能
- 點擊左右箭頭切換
- 點擊底部導航點
- 使用鍵盤方向鍵
- 滑鼠懸停暫停自動播放

### 4. 瀏覽賽事
- 點擊日期選擇不同日期
- 左右滑動日期導航
- 點擊賽事卡片查看詳情
- 使用功能圖示（直播、購票、統計）

### 5. 搜尋功能
- 在搜尋欄輸入關鍵字
- 按 Enter 搜尋
- 即時顯示搜尋結果

### 6. 主題切換
- 點擊右上角太陽/月亮圖示
- 即時切換日間/夜間模式
